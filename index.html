C-BIOS Association

What is C-BIOS?

  • C-BIOS is a BIOS compatible with the MSX BIOS
  • C-BIOS was written from scratch by BouKiChi
  • C-BIOS is available for free, including its source code (2-clause BSD license)
  • C-BIOS can be shipped with MSX emulators so they are usable out-of-the-box without copyright issues

Plan to accelerate C-BIOS development

  • Make C-BIOS is downloadable again. [Done]
  • Discuss this plan and improve it. When it is ready, do a public announcement.
  • Create a project on SourceForge.net for C-BIOS. [Done]
  • Create mailing list for discussion among developers. [Done]
  • Create a bug tracker, where users can report cases when C-BIOS is not yet compatible.
  • Make a list of emulators shipping C-BIOS already (at least openMSX and blueMSX). Mail authors of other emulators to tell them about C-BIOS.
  • Decide what assembler syntax to support.
  • Invite developers to join C-BIOS development. [In progress]
  • Set up a debug and test environment, in which bug reports can be analysed efficiently. Also spot calls to non-implemented functions.
  • Development of a Disk ROM. Start with documenting the interface of a Disk ROM (entry points, hooks, system variables). Then implement Disk ROM for one particular FDC.
